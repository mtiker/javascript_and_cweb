# JavaScript Assignment 01 - Task Manager

## Objective

Build a browser-based task management utility using pure JavaScript.

## Commands

- add
- list
- update
- delete
- filter
- search

## Data Model

- id
- title
- description
- status
- priority
- dueDate
- tags[]

## Notes

- Use browser storage (`localStorage` or `IndexedDB`).
- Keep async operations in `try/catch`.
- Validate all inputs before saving.
