# JavaScript Assignment 01 Submission Checklist

## Functionality (40)

- [ ] add command creates a task
- [ ] list command shows all tasks
- [ ] update command edits selected task
- [ ] delete command removes selected task
- [ ] filter command works by status/priority/due date/tag
- [ ] search command finds title/description/tags
- [ ] data persists after refresh (localStorage)

## Code Quality (25)

- [ ] modular structure (`ui`, `service`, `storage`, `validation`)
- [ ] clear naming and separation of concerns
- [ ] no duplicated core logic

## Error Handling (15)

- [ ] async operations wrapped in safe handlers
- [ ] storage errors handled with clear messages
- [ ] validation errors shown to user
- [ ] corrupted storage data does not crash app

## Git Usage (10)

- [ ] commit history shows feature progression
- [ ] meaningful commit messages

## Documentation (10)

- [ ] assignment README has run/usage details
- [ ] AI-assisted evidence in `docs/ai-prompts.md`
